Lilypad Help & How To

Create New Pad

  • Click the "New Pad" in the button right corner of the Home Page
  • Name your pad before you start adding files, URLs, and standalone programs (please)
  • Add the files, URLs, and standalone programs you want
  • Choose a color for your pad (the default color is red)
  • Click the "Done" button in the right corner of the New Pad page
  • Now you have created a beautiful Pad!

Delete a Pad

  • Click "Edit Pad" on the pad you wish to delete
  • Click the "Delete Pad" button in the bottom left corner of the "Edit Pad" page
  • Choose yes if you are sure you you want to delete this entire pad

Edit Pad

  • Place your cursor over the pad you wish to edit
  • "Edit Pad" and "Launch Pad" buttons will appear
  • Click "Edit Pad"
  • You can delete contents, add contents, change the launch program of contents, change the name, and change the color

Launch Pad

  • Place your cursor over the pad you wish to launch
  • "Launch Pad" and "Edit Pad" buttons will appear
  • Click "Launch Pad"

Add Files

  • This can be done on the "New Pad" and the "Edit Pad" pages
  • Click the "add some files" button at the bottom in the middle of the screen
  • A file browser will appear
  • Select the files you wish to add
  • Click "done" in the file browser
  • The selected files were added to your Pad's contents

Add URLs

  • This can be done on the "New Pad" and the "Edit Pad" pages
  • Click the "add a website..." textbox
  • Type or paste a URL
  • Press the enter key on your keyboard or the white arrow button next to the URL you've entered

Change launch program

  • For all files or URLs under a given launch program

    • This can be done on the "New Pad" and the "Edit Pad" pages
    • Click the chevron (v) button next to the launch program you wish to change
    • A list of alternative programs will appear
    • Select the program you wish to use to launch all files or URLs
    • The program has now been changed

  • For one file or URL

    • This can be done on the "New Pad" and the "Edit Pad" pages
    • Hover over the file or URL for which you wish to change the launch program
    • Click the chevron (v) button which appears next to the file or URL
    • A list of alternative programs will appear
    • Select the program you wish to use to launch the file or URL
    • The program has now been changed

Change Pad Color

  • This can be done on the "New Pad" and the "Edit Pad" pages
  • The default pad color is red
  • Choose one of the five colors next to the title entry box (please)
  • This color is now the Pad color
